## 六、代码块 
`for (int i = 10; i >= 0; i--){`

```c++
for (int i = 10; i >= 0; i--){
    std::cout << "i=" << i << std::endl;
}
for (unsigned j = 10; j >= 0; j--){
    std::cout << "j=" << j << std::endl;
}
```
